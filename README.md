## 📂 Current Highlight: Metropolis Government Portal
My most recent project involved building a public service portal from scratch using pure HTML5.

**Key Technical Achievements:**
* **Semantic Structure:** Utilized `<header>`, `<nav>`, `<section>`, and `<footer>` for SEO and accessibility.
* **Complex Forms:** Implemented 3-column `<fieldset>` groupings with strict data validation (tel, email, maxlength).
* **Information Architecture:** Organized government documentation and news articles using nested lists and `<article>` tags.
