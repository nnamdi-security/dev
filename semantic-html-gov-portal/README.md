# Metropolis Government Public Services Portal

A semantically structured, HTML5-only public service portal developed as a training group assessment. This project demonstrates a core understanding of web accessibility, document hierarchy, and form validation without the use of CSS or JavaScript.

## 🚀 Project Overview
This portal serves as a central hub for citizens to access public documents, stay updated with local government news, and submit service requests.

### Key Features
* **Semantic Header & Navigation:** Clear site hierarchy with emergency contact highlights.
* **Public Documents Section:** An organized, downloadable list of official resources using ordered lists.
* **Government News:** standalone articles featuring `<time>` elements and official citations.
* **Citizen Services Request Form:** A complex, accessible form utilizing `<fieldset>`, `<legend>`, and various input types (tel, date, file, etc.).
* **Three-Column Footer:** Comprehensive site map and contact information using `<address>`.

## 🛠️ Technical Requirements Met
- [x] Use of HTML5 semantic tags (`<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`).
- [x] Advanced Form structures with labels and fieldsets.
- [x] Implementation of the `<time>` and `<cite>` elements for metadata.
- [x] Proper document nesting (Ordered and Unordered lists).
- [x] Use of the `maxlength` and `required` attributes for basic validation.
